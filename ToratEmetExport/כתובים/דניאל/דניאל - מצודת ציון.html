פרק-א

א ויצר. מלשון מצור וענינו הקפת החיל על העיר להציר להם ולכבשם:

ב שנער. היא בבל ע''ש שננערו בה מתי המבול כי היא עמוקה מכל העולם:

ג סריסיו. שריו: הפרתמים. ענין שררה ועושר וכן חיל פרס ומדי הפרתמים ( אסתר ח ) והיא מלה זרה ואולי היא מלשון כשדים וכמוהו רבים ימצאו בזה הספר אשר אין לדעת ביאורם כ''א לפי הקרוב לענינם:

ד מאום. כמו מום: מדע. ענין מחשבה כמו גם במדעך מלך אל תקלל ( קהל' י ): ספר. ר''ל כתב הנכתב בספר:

ה וימן. ענין הזמנה כמו וימן ה' דג ( יונה ב ): מפת בג. ענין מאכל:

ח יתגאל. ענין לכלוך וטנוף כמו לחם מגואל ( מלאכי א ):

י מנה. הזמין: זועפים. ר''ל רעים והוא מל' זעף וכעס כי הכועס נשתנה מראית פניו לרוע: כגילכם. כדמותכם ובדרז''ל בן גילו ( יבמות קכ ):

יא המלצר. הוא המסדר המאכל: אשר מנה. אשר הפקיד כמו ומהם ממונים על הכלים ( דה''א ט' ):

יב נס. מלשון נסיון: הזרועים. מיני זרעונים כקטניות וכיוצא:

טו ובריאי. ענין שומן כמו ועגלון איש בריא ( שופטים ג ):

יז חזון. ראיית החלום והוא כפל ענין במ''ש:

כ ידות. חלקים כמו וארבע הידות ( בראשי' מז ): החרטומים. הם חכמי הטבע. האשפים. הם חכמי הרפואה:

פרק-ב

א ותתפעם. ענין הכאה וקשקוש וכן ויהי בבוקר ותפעם רוחו ( בראשי' מ''א ). והוא מלשון פעמון המקשקש: ושנתו. מלשון שינה: נהיתה. ענין כריתה ושבירה כמו נהייתי ונחליתי ( לקמן ח ):

ד ופשרא. ופתרונו כמו ופותר אין אותו ( בראשי' מ ) תרגומו ופשר לית ליה וכן רוב הספר הוא לשון תרגום והוא ארמית ואין צורך להביא ראיה עליהם:

ה אזדא. הלכה ובדרז''ל ואזדא לטעמיה: הדמין. בת''י הוא ענין נתחים:

ו ונבזבה. ענין דורון גדול בויתור רב ובדרז''ל המבזבז אל יבזבז וכו' ( כתובות נ ):

ח יציב. ענין אמת וכמוהו רבים בזה הספר ובתפלה אמת ויציב והוא מלשון דבר הניצב ועומד ומתקיים:

יב בנס. בזעף כי והנם זועפים ( בראשית מ ) תרגומו נסיסין:

טו ענה. ענינו הרמת קול כמו וענו הלוים וגו' קול רם ( דברים כז ): מהחצפה. ממהרת והוא מל' חצוף הממהר להביש ולהחפיר את הבריות:

כו כהל. יכול וכן ולא כהלין כתבא למקרא ( לקמן ה ):

כז גזרין. הם בעלי הקסם ההולכים לקראת נחשים:

כט רעיוניך. מחשבתך כמו בנתה לרעי ( תהלים קלט ):

ל על דברת. ענינו כמו בעבור וכן על דברת בני האדם ( קהלת ג ):

לא ואלו. והנה וכן ואלו עיר וקדיש ( לקמן ד ): דכן. זה כי מי האיש הלזה ( בראשית כד ) ת''א דיכי:

לב וירכתה. שלשה פרקים יש ברגל העליון קרוי ירך האמצעי קרוי שוק התחתון קרוי רגל:

לה כעור. כמוץ או יתכן שתהיה מל' נעורת של פשתן ובא בחסרון הנו''ן ור''ל כדבר אשר ינוער:

מ וחשל. ומרדד ובדרז''ל חשלי דודי ( כתובות עז ):

מא די פחר. של יוצר חרס ובדרז''ל במני דפחר' ( תענית ז ): נצבתא. ענין נטיעה כמו מצבת בם ( ישעיה ו ) ור''ל הכח והיכולת ע''ש שהנטיעה היא כח הפירות והעלין:

מח סגנין. ענין שררה מה כמו ויבא סגנים כמו חומר ( ישעיה מא ):

פרק-ג

ב לאחשדרפניא וכו'. כולם שמות מיני שררה זו למעלה מזו וידועים המה בל' כשדים: לחנכת. התחלת תשמיש הדבר להיות משמש והולך קרוי חנוך בל' המקרא:

ד וכרוזא. ענין השמעת הדבר לרבים: עממיא. הם האומות שיש להם אלוה ותורה: אומיא. הם בלא אלוה ובלא תורה: ולשניא. הם החלוקים בלשונותם בין מבני אלוה ותורה ובין זולתם:

ה משרוקיתא וכו'. כולם שמות מיני כלי זמר וידועים המה בל' כשדים:

ח ואכלו קרציהון. הוא מענין רכילות ומלשינות כי לא תלך רכיל ( ויקרא יט ) ת''א לא תיכול קורצין ע''ש שדרך המלשין לרמז דברו ולקרוץ בעיניו למען לא יבינו כל העומדים ובהקריצה ההיא כאלו אוכל בשר מי אשר ילשין עליו:

י טעם. ענינו בכ''מ דבר שיש בו ממש הנעשה בטעם ולא דבר של מה בכך:

יד הצדא. ענין שממה כי תהו ובהו ( בראשית א ) ת''א צדיא ורקניא:

טז חשחין. כמו חוששין הידוע בדרז''ל:

יט למזא. ענין הסקה ושריפה וכן מזי רעב ( דברים לב ):

כ לכפתה. ענין קשירה ובדרז''ל עבד כפות לו ( ב''ק כב ):

כא בסרבליהון. הוא שם מלבוש ובדרז''ל בסרבלי חתימי ( שבת נח ): פטשיהון. הם המכנסים: וכרבלתהון. מין לבוש עשוי להתעטף בו ודוגמתו ודוד מכורבל במעיל ( דה''א טו ):

כב שביבא. ניצוצות כמו ולא יגה שביב אשו ( איוב יח ):

כד להדברוהי. הם היועצים המנהיגים כי וינהג את הצאן ( שמות ג ) ת''א ודבר:

כה לבר אלהין. כן יקרא המלאך כאלו הוא מבני בית של השכינה וכן ויבואו בני האלהים ( איוב א ):

כז בגשמיהון. בגופם כי ושבט לגו כסילים ( משלי כו ) ת''י ושבטא לגושמיהון: התחרך. ענין שריפה כמו לא יחרוך רמיה ( שם יב ): עדת. עברה כמו לא עדה עליו שחל ( איוב כח ):

כט שלו. שגגה ומשגה כמו על השל ( ש''ב ו ):

לא ישגא. יתגדל ויתרבה כמו כארז בלבנון ישגה ( תהלים צב ):

לב שפר. נאה ויאה כמו אמרי שפר ( בראשית מט ):

לג כמה. הוא כענין מאד ובדרז''ל כמה גדולים מעשה חייא ( כתובות קג ):

פרק-ד

א ורענן. ענין לחות ורטיבות כמו דשנים ורעננים ( תהלים צב ):

ו אנס. ענין הכרח ונצוח ועל שם זה נקרא הגזלן אנס:

ט עפיה. העלין כמו מבין עפאים ( שם קד ):

י עיר. מלאך המחריב והוא מלשון ערו ערו ( שם קלז ):

יא גדו. כרתו והוא קרוב מלשון שקמים גדעו ( ישעיה ט ): אתרו. ענין הפלה והשלכה כמו ראה ויתר גוים ( חבקוק ג ):

יד שאלתא. הוא ענין גזירה ע''ש כי נעשה בשאלת פי ה': עד דברת. כמו על דברת ור''ל בעבור:

טז אשתומם. ענין תמהון ושתיקה כמו משמים בתוכם ( יחזקאל ג ): לערך. לשונאך כמו ויהי ערך ( ש''א כה ):

כב יטעמון. מלשון טעימה ואכילה:

כד מלכי. עצתי כמו וימלך לבי עלי ( נחמיה ה ):

לג יבעון. ענין שאלה ובקשה כמו אם תבעיון בעיו ( ישעיה כא ):

לד בגוה. כמו בגאוה ודוגמתו כי השפילו ותאמר גוה ( איוב כב ):

פרק-ה

א לחם רב. כל סעודה קרוי' ע''ש הלחם:

ב שגלת'. היא המלכה המיוחדת לו למשכב המשגל:

ה נברשתא. מנורה ובדרז''ל עשתה נברשת של זהב ( יומא לז ): גירא. סיד כמו כאבני גיר ( ישעיה כז ): פס ידא. כף היד ובדרז''ל שלשים בפיסת היד ( אהלות פ''א ):

ו נקשן. ענין הכאה כמו מקשה תיעשה המנורה ( שמות כה ):

ז ארגונא. ארגמן וכן ובארגון וכרמיל ( דה''ב ב ) והוא שם מין צבע חשוב: והמניכא. ענק רצוף בטבעות לתכשיט כי רביד הזהב ( בראשית מא ) ת''א מניכא דדהבא:

ט משתבשין. ענין בלבול ושבוש הדעת:

יא נהירו. הארה כמו ואל תופע עליו נהרה ( איוב ג ):

יז ונבזביתך. ענין דורון גדול כמו מתנן ונבזבה ( לעיל ב ):

יט זייעין. ענין חרדה ורתת כמו והיה רק זועה ( ישעיה כח ):

כא ערדיא. פראים כמו ומוסרות ערוד ( איוב לט ):

כד רשים. ענין כתיבה וכן את הרשום בכתב ( לקמן י ):

כח פרס. ענין שבירה כמו הלא פרוס לרעב לחמך ( ישעיה נח ):

פרק-ו

ג סרכין. שם מין שררה:

ד עשית. ענין מחשבה כמו אבדו עשתונותיו ( תהלים קמו ):

ה עילה. עלילה, ובדרז''ל עילה מצאו ( כתובות כ ): שלו. משגה כמו על השל ( ש''ב ו ):

יב הרגשו. ענין קבוץ כי בקבוץ עם רב נשמע קול המיה ורגישה וכן למה רגשו גוים ( תהלים ב' ):

טו בל. הוא מל' מנדה בלו והלך ( עזרא ד' ) וענינה מס וגולגולת, או הוא מענין לאו כמו בל למד צדק ( ישעיה כו ): משתדר. ענין ספק והשתדלות בדרז''ל השתדל להיות איש ( אבות פ''ב ):

יט ובת טות. לן מבלי סעודת הערב וכן בדרז''ל ובת טות ולא הבדיל ( פסחים קז ) ר''ל לא אכל בלילה על שלא הבדיל: ודחון. הוא הפוך מן חדון ור''ל שמחת כלי זמר:

כ בשפרפרא. כן ת''י על השחר: בנגהא. מל' נוגה ואורה:

פרק-ז

ב וארו. כמו ואלו ר''ל והנה: מגיחן. ענין יציאה כמו גוחי מבטן ( תהלים כב ):

ד מריטו. תלשו כמו ואמרטה משער ראשי ( עזרא ט ):

ה ולשטר. צלע ת''א סטר:

ו באתר. כמו בתר בלא אל''ף:

ז רפסה. רמסה כמו מתרפס ברצי כסף ( תהלים סח ):

ט ועתיק. ענין מיושן כמו והדברים עתיקים ( דה''א ד' ):

י ורבו. הוא עשרת אלפים:

יב וארכה. ענין המתנה ואריכת זמן: זמן ועידן. הם שמות ( יד ) וליה. ונתן לו הממשלה והכבוד והמלוכה וכל העכו''ם יעבדו לו: נרדפים וכן אדמת עפר ( לקמן יב ):

טו נדנה. ענין תיק ונרתק כמו וישב חרבו אל נדנה ( דה''א ד ):

כג ותדושנה. מל' דישה:

כה יבלא. מל' בליה והפסד: ויסבר. ענין מחשבה ובדרז''ל סברוהו רבנן: עדן. כן התרגום של עת ושל שנה:

פרק-ח

ב הבירה. ענין ארמון וכן לא לאדם הבירה ( שם כט ) ור''ל שהיתה מטרפולין מקום הארמונות: אובל. נהר והוא מל' יבלי מים ( ישעיה ל ) ובא האל''ף במקום היו''ד כמו ישי ואישי ( דה''א ב ): אולי. כן שם הנהר:

ד מנגח. ענין הכאה ודחיפה בקרן:

ה צפיר. הוא שעיר וכן צפירי חטאת ( עזרא ח ): חזות. ענין ראיה ומראה:

ט מצעירה. קטנה כמו והיא מצער ( בראשית יט ): הצבי. החמודה וכן נחלת צבי ( ירמיה ג ):

יא הורם. ל' הרמה וסלוק: מכון. מל' כן ובסיס:

יב וצבא. זמן קצוב כמו הלא צבא לאנוש ( איוב ז' ):

יג לפלמוני. היא מלה מורכבת מפלא ואלם ופלא הוא ענין מכוסה כמו כי יפלא ממך ( דברים יז ) ואלם ר''ל לא ידובר בו ועל כי היה שמו נעלם ממנו אמר לפלמוני: שומם. משותק כמו משמים בתוכם ( יחזקאל ג ): וקדש. וי''ו יתירה: . ( טז ) להלז. להזה כמו השונמית הלז ( מ''ב ד' ):

יז נבעתי. מל' בעתה וחרדה: כי לעת קץ. אשר לעת קץ:

יח נרדמתי. מלשון תרדמה ושנה:

כא והצפיר השעיר. כפל המלה בשמות נרדפים לתוספת ביאור:

כג כהתם. מלשון תם והשלמה: עז פנים. חצוף:

כו כי לימים רבים. אשר לימים רבים:

כז נהייתי. נשברתי וכן ושנתו נהיתה עליו ( לעיל ב' ):

פרק-ט

ב בספרים. מלשון מספר וחשבון:

ד אנא. ענין לשון בקשה:

ז במעלם. ענין חטא ופשע:

יא ותתך. ענין יציקה והרקה כמו ומטר לא נתך ארצה ( שמות ט ): האלה. הקללה:

יג חלינו. ענין תפלה כמו ויחל משה ( שם לב ):

יד וישקד. ענין מהירות כמו כי שוקד אני ( ירמיה א ):

יח פקח. פתח:

יט הקשיבה. ענין שמיעה: תאחר. מל' איחור והעכבה:

כא מועף ביעף. מל' עפיפה ופריחה: מנחת ערב. תשורת ערב:

כד שבועים. שמטות משבע שנים: נחתך. נגזר כי שניהם הם מענין כריתה: ולהתם. מל' תם והשלמה: ולחתום. ענין גמר ובדרז''ל כל חותמי ברכות ( ברכות סג ) וכן ועל החתום ( נחמיה י ) כי דרך לרשום כ''א שמו אחר גמר הכתב: ולמשוח. ענין גדולה כמו למשיחי לכורש ( ישעיה מה ) ולזה קראו גם כאן משיח נגיד:

כה וחרוץ. ענין חפירה ובדרז''ל חריצים ונעיצים ( חולין סז ): ובצוק. מלשון צוקה וצרה:

כו בשטף. הוא לשון מושאל משטיפת מרוצת המים: נחרצת. ענין כריתה וחתוך וכן כי כלה ונחרצה ( ישעיה י' ):

כז לרבים. לשרים כמו ורבי המלך ( ירמיה מ''א ): משומם. משותק ונאלם: תתך. ענין יציקה והרקה:

פרק-י

א וצבא. ענין חיל רב:

ג עד מלאת. עד נשלם:

ה בדים. פשתן: בכתם. ענין זהב וכן וחלי כתם ( משלי כ''ה ):

ו כלפידי. כלהבות: ומרגלותיו. רגליו: כעין. כגוון כמו ועינו כעין הבדולח ( במדבר יא ): קלל. מזוקק באש והוא מל' קלוי באש ( ויקרא ב ):

ח עצרתי. ענין עכבה כמו ולא תעצור כח הזרוע ( לקמן י''א ):

י ותניעני. מל' נענוע:
