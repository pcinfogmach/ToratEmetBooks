פרק-א

ב אסוף אסף. ענין כליון כמו ואספת את נפשך ( שופטים יח ) והיה ראוי לומר אסף אאסף ותחס' האל''ף:

ג את הרשעים. עם הרשעים:

ד שאר. שיור: הכמרים. כן יקראו כהני עבודת כוכבים וכן וכמריו עליו יגילו ( הושע י ): הכהנים. הם מין אחר מתחלפים בלבושיהם:

ו הנסוגים. ענין החזרה לאחור כמו ולא נסוג ממך ( תהלים פ ):

ז הס. ענין שתיקה כמו ויהס כלב ( במדבר יג ): הקדיש. ענין הזמנה כמו קדשו קהל ( יואל ב ): קרואיו. המזומנים לסעודה וכן בראש הקרואים ( ש''א ט ):

ח ופקדתי. ענין השגחה: נכרי. ענין זרות:

ט הדולג. ענין קפיצה: המפתן. אסקופה כמו מתחת מפתן הבית ( יחזקאל מז ): חמס. עושק:

י המשנה. מל' שנים ור''ל בין שתי החומות שהיא משנה לעיר וכן והיא יושבת בירושלים במשנה ( מ''ב כב ):

יא הילילו. מל' יללה: המכתש. שם מקום בירושלים עמוקה ככלי המכתשת שדוכין בה בשמים: נדמה. ענין כריתה כמו ער מואב נדמה ( ישעיה טו ): עם כנען. ר''ל עדת התגרים כמו כנען בידו מאזני מרמה ( הושע יב ): נטילי. ענין משא כמו ונטל החול ( משלי כו ):

יב אחפש. ענין בקור ודרישה כמו תמנו חפש מחפש ( תהלים סד ): הקופאים. הצפים ממעל כי ויצף הברזל ( מ''ב ו ) תרגומו וקפא פרזלא: שמריהם. הוא עכירת המשקה הנופל לשולי הכלי כמו ושקט הוא אל שמריו ( ירמיה מח ): ירע. מלשון רעה:

יג חילם. עשרם כמו ישא את חיל דמשק ( ישעיה ח ): למשסה. ענין רמיסה כמו למשסה ואין אומר השב ( שם מב ):

יד צורח. ענין שאגת קול גדול כמו יריע אף יצריח (, שם ):

טו עברה. ענין כעס: ומצוקה. עניינו כמו צרה וכן בארץ צרה וצוקה ( שם ל ) וכפל המלה בשמות נרדפים להפלגות הענין: שואה ומשואה. ענין חושך ושממון כמו אמש שואה ומשואה ( איוב ל ): וערפל. הוא עב הענן:

טז הפנות. הם המגדלים שדרכם לעמוד בפנות וזויות המבצר:

יז והצרותי. מל' צרה: ולחומם. כן יקרא גם הבשר כמו כי לחמם לנפשם ( הושע ט ): כגללים. כצואה כמו כגללו לנצח יאבד ( איוב כ ):

יח קנאתו. ענין כעס וחמה כמו הם קנאוני בלא אל ( דברים לב ): תאכל. ענין השחתה: נבהלה. מל' בהלה:

פרק-ב

א התקוששו וקושו. ענין החפוש והלקיטה היטב כמו לקושש קש ( שמות ה' ): נכסף. ענין חשק וחמוד כמו נכסוף נכספתה ( בראשית לא ):

ב חוק. גזירת המקום: כמוץ. היא פסולת התבואה כמו כמוץ יסוער מגורן ( הושע יג ):

ג בקשו. ענין דרישה:

ה חבל. כן יקרא המחוז כמו חבלי מנשה ( יהושע יז ):

ו נות. מל' נוה ומדור: כרות. ענין חפירה כמו בור כרה ( תהלים ז ): וגדרות. בנין גדר למרבץ צאן:

ז ירעון. מל' מרעה: ירבצון. ענין השכיבה לנוח: ושב. ענין ההשקט ומרגוע כמו בשובה ונחת ( ישעיה ל ): שביתם. מל' שבי הגולה:

ח וגדופי. ענין חרפה כמו וישראל לגדופים ( שם מג ):

ט ממשק. ענין השמעת קול כמו כמשק גבים ( שם לג ): חרול. מין קוצים כמו תחת חרול יספחו ( איוב ל ): ומכרה. מל' כרייה וחפירה: יבזום. מל' בזה:

י גאונם. מל' גאוה:

יא רזה. ענין כחשות כמו במשמניו רזון ( ישעיה ו ):

יג ציה. ענין חורב ויובש:

יד חיתו. גם הבהמה תקרא בשם חיה וכן וחיתם לדבר הסגיר ( תהלים עח ): קאת. קפוד. שמות עופות המצויות בחרבות: בכפתוריה. כן יקרא המשקוף לפי שעושין בו ציורי כפתורים וכן הך הכפתור ( עמוס ט ): ילינו. ענין התמדה וכן צדק ילין בה ( ישעיה א ): ישורר. מל' שיר ורנן: חרב. מל' חורבן: בסף. מזוזה כמו וסף השער ( יחזקאל מ ): ארזה. תקרת הארז: ערה. ענין גלוי כמו וקיר ערה מגן ( ישעיה כ''ב ):

טו העליזה. ענין שמחה כמו יעלזו חסידים ( תהלים קמט ): ואפסי. כמו ואין: ישרוק. ענין השמעת קול הבאה בקבוץ השפתים כמו שרקו ויניעו ראשם ( איכה ב' ): יניע. מלשון הנעה והנדה:

פרק-ג

א מוראה ונגאלה. ענין שניהם מלוכלך ומטונף כמו ושמתיך כרואי ( נחום ג ) וכמו נגאלו בדם ( ישעיה נט ) וכפל המלה בשמות נרדפות: היונה. מל' אונאה ועניינו קחת ההון וכן מפני חרון היונה ( ירמיה כה ):

ג גרמו. ענין עצם כמו גרמיו כמטיל ברזל ( איוב מ ):

ד פוחזים. קלי הדעת וכן אנשים ריקים ופוחזים ( שופטים ט ): בוגדות. מל' בוגד: חללו. ענין בזיון: חמסו. ענין הסרה כמו ויחמוס כגן שכו ( איכה ב ):

ה לאור. ר''ל ברור כאור ודומה לו: נעדר. ענין חסרון כמו איש לא נעדר ( ישעיה מ ):

ו נשמו. מל' שממון: פנותם. הם המגדלים הגבוהים כמו ועל הפנות הגבוהות ( לעיל א ): החרבתי. מל' חרבן: חוצותם. ענין שוק ורחוב: נצדו. ענין שממון כי והשמותי אני את הארץ ( ויקרא כו ) ת''א ואצדי:

ז אך. עניינו כמו באמת וכן אך עצמי ובשרי אתה ( בראשית כט ): מעונה. מדורה כמו ה' מעון אתה היית ( תהלים צ ): פקדתי. צויתי כמו והוא פקד עלי ( עזרא א ): עלילותם. מעשיהם:

ח חכו. ענין המתנה ואיחור כמו ונסתה ולא תחכה ( מ''ב ט ): לעד. ענין בזה ושלל כמו בבוקר יאכל עד ( בראשית מט ):

ט שפה. כן יקרא לשון עם ועם וכן שפת רעהו ( שם יא ): ברורה. כלפי שקורא לשון העכו''ם עמקי שפה כמ''ש עם עמקי שפה ( ישעיה לג ) קורא למולם לשון הקדש שפה ברורה ונקיה: שכם. ר''ל צד אחד ודעה אחת כמו דרך ירצחו שכמה ( הושע ו ):

י עתרי. שם אומה: בת פוצי. עדת פוצי והוא שם אומה: יובילון. ענין הבאה כמו יובל שי ( ישעיה יח ):

יא עלילותיך. מעשיך: עליזי. ענין שמחה:

יב עני ודל. ר''ל מוכנע כעני ודל: וחסו. ענין בטחון:

יג תרמית. מל' רמיה ומרמה: ירעו. מל' מרעה: ורבצו. ענין השכיבה לנוח:

טו פנה. ענין הסרה וביעור כמו פנית לפניה ( תהלים פ ):

טז ירפו ידיך. כן נקרא המתפחד ומתייאש וכן ורפה ידים ( ש''ב יז ):

יז יחריש. ענין שתיקה כמו והחריש לה ( במדבר ל ):

יח נוגי. מל' יגון ותוגה: ממועד. ענין זמן ועת כמו למועד אשוב אליך ( בראשית יח ): אספתי. ענין הכנסה כמו ואין איש מאסף אותם ( שופטים יט ): משא. מל' משא וסבל:

יט עושה. ענין כתישה ומעוך כמו בעשות ממצרים דדיך ( יחזקאל כג ): מעניך. מל' עינוי: הצולעה. החגרת כמו והוא צולע על ירכו ( בראשית לב ): והנדחה. מלשון דחיה: לתהלה. מל' הלול ושבח: לשם. ר''ל לפרסום שם טוב:

כ בשובי. ענין השקט ומרגוע כמו בשובה ונחת ( ישעיה ל ): שבותיכם. מל' שבי הגולה: וסגנים ( יחזקאל כג ):

פרק-ד

ד ספונים. ענין תקרת הכסוי כמו וספון בארז ( ירמיה כב ): חרב. מלשון חורבן:

ה דרכיכם. ר''ל עסקיכם
