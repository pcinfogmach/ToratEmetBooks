פרק-א

ה אדוני בזק. כך שמו, וכן ( יהושע י א ) אדוני צדק:

ו ויקצצו. ענין חתוך וכריתה, כמו ( שמות לט ג ) וקצץ פתילם: בהנות. אצבעות אגודל, כמו ( שם כט כ ) בהן ידם הימנית:

ז מלקטים. לשון לקיטה:

ח שלחו. על כי אם מי יבעיר האש באחד מן המקומות, עשה יעשה שליחותו, והולך ממקום למקום ושורף הכל, לזה נקרא בלשון שלוח:

ט והשפלה. העמק:

י לפנים. בימים הקדמונים:

יד ותסיתהו. מלשון הסתה ופתוי: ותצנח. ענין נעיצה, כמו ( שופטים ד כא ) ותצנח בארץ. ורוצה לומר, נפלה בכח, וכאלו ננעצה בארץ:

טו הבה. תנה, כמו ( בראשית מז טו ) הבה לנו לחם: ברכה. מתנה ומנחה, וכן ( שם לג יא ) קח נא את ברכתי: הנגב. ענין יובש, כמו ( ישעיהו כא א ) כסופות בנגב: נתתני. נתת לי: גלת. מעין, כמו ( שיר השירים ד יב ) גן נעול: עלית. מלשון עליון:

יז ויחרימו. ענין אבדון וכליון:

יט ויורש. מלשון גרושין: רכב ברזל. אנשי רכב, חזקים כברזל:

כ הענק. אנשים גבוהי הקומה:

כג ויתירו. ענין חקירה וחפוש, כמו ( במדבר יג ב ) ויתרו את ארץ כנען:

כד מבוא. מלשון ביאה:

כז ויואל. ענין רצון, וכן ( שמות ב כא ) ויואל משה:

לד וילחצו. ענין דחק, כמו ( במדבר כב כה ) ותלחץ אל הקיר: לא נתנו. לא הניחו, כמו ( בראשית לא ז ) ולא נתנו אלהים:

לה ותכבד. עניינו כמו ותחזק:

פרק-ב

א מלאך. ענין שליח: הבכים. אמר על שם סופו, שקראו המקום 'בכים': אפר. מלשון הפרה ובטול:

ב תתצון. מלשון נתיצה ושבירה:

ג צדים. לשון צדיה, ורצונו לומר לצודד נפשות:

ט חרס. הוא סרח האמור בספר יהושע ( כב ל ) ומתהפך הוא, כמו כבש כשב:

י נאספו. ענין מיתה, כמו ( במדבר כ כד ) יאסף אהרן:

יג לבעל ולעשתרות. שמות עבודה זרה:

יד שסים. דורכים ורומסים, כמו ( זכריה יד ב ) ונשסו הבתים: וימכרם. מסרם, וכאלו מכרם:

טו ויצר. מלשון צר ודוחק. ( יז ) זנו. המשילם לזונה העוזבת בעל נעוריה, ותאהב זולתו:

יח ינחם. ענין הפוך מחשבה: מנאקתם. ענין צעקה מכאב לב, כמו ( שמות ו ה ) שמעתי את נאקת: לחציהם ודחקיהם. הוא כפל ענין במלות שונות ובא לחזק הדבר, והוא כלל גדול בדברי הנביאים:

יט לא הפילו. לא גרעו, ודוגמתו ( איוב יב ג ) לא נופל אנכי מכם: ממעלליהם. ממעשיהם:

כ ויחר. ענין כעס: יען. בעבור:

כא להוריש. לגרש: עזב. הניח:

כב נסות. מלשון נסיון:

פרק-ג

ג סרני. ענין שררה:

ז אשרות. אילן הנעבד:

י ותעז. ענין חוזק:

יא ותשקט. ענין מנוחה:

טו אטר. ענין סגירה, כמו ( תהלים סט טז ) ואל תאטר עלי באר פיה. ורצה לומר: כאלו יד ימינו סגורה, כי לא שלט בה:

טז פיות. חודה של חרב נקראת פה: גמד. אמה, וכן ( יחזקאל כז יא ) גמדים במגדלותיך, שרצה לומר, אנשי מדה קצרה: למדיו. למלבושיו, כמו ( ויקרא ו ג ) מדו בד: ירך. צד, כמו ( ויקרא א יא ) ירך המזבח:

יז בריא. בעל בשר שמן, וכמו ( מלכים א ה ג ) עשרה בקר בריאים:

יט הפסילים. מקום שמחליקים שם פני האבנים, כמו ( שם שם לב ) ויפסלו בני שלמה: הס. ענין שתיקה, כמו ( במדבר יג ל ) ויהס כלב: מעליו. מאצלו:

כ בעלית. מלשון עליה, הבנויה ממעל: המקרה. מלשון קור וצינה:

כא ויתקעה. ענין נעיצה ותחיבה, וכן ( שמואל ב יח יד ) ויתקעם בלב אבשלום:

כב הנצב. הוא יד החרב שמעמיד אותה לעשות מלאכתה: הלהב. היא ברזל החרב, כי היא מברקת כלהב כשהיא מרוטה, כמו ( נחום ג ג ) ולהב חרב: שלף. ענין הוצאה, כמו ( יהושע ה יג ) וחרבו שלופה: הפרשדנה. הפרש והצואה:

כג המסדרונה. היא האכסדרה המסודרת בכסאות לישיבת אנשים: בעדו. כנגדו, כמו ( בראשית ז טז ) ויסגור ה' בעדו: ונעל. רוצה לומר, גם סגרה בהמנעול:

כד מסיך. הוא כנוי לעושה צרכיו, כמו ( שמואל א כד ג ) ויבא שאול להסך את רגליו, על שם שדרך היושב להפנות, לסכך ולכסות רגליו מחמת צניעות:

כה ויחילו. המתינו, כמו ( בראשית ח י ) ויחל עוד: בוש. ענין איחור, כמו ( מלכים ב ח יא ) וישם עד בוש: המפתח. שם הכלי הפותח המנעול:

כו עד. כמו בעוד: התמהמהם. ענין עכבה, כמו ( בראשית מג י ) לולא התמהמהנו: השעירתה. שם מקום:

ל ותכנע. מלשון הכנעה:

לא במלמד הבקר. הוא המקל שבראשו, תקועה כעין מחט הנקרא דרבן, ובו מלמד ומורה הבקר לחרישה:

פרק-ד

ג ברזל. חזקים כברזל:

ו מקדש נפתלי. שם מקום בנחלת נפתלי:

ז המונו. הוא ענין עם רב, על שם שקול המייתם רבה:

ט אפס. רק: תפארתך. שבחך: ימכור. ימסור:

י ויזעק. ענינו אסיפה הבאה בזעקת המאסף, וכן ( לקמן יח כג ) מה לך כי נזעקת:

יד קום. הוא ענין זרוז:

טו ויהם. מלשון מהומה:

יח סורה. ענין נטיה מן המקום: בשמיכה. היא כסות עב ויוצאים ממנה ציציות, וכתרגום יונתן: בגונכא, וכן ( מלכים ב ח טו ) ויקח המכבר, תרגם יונתן: גונכא:

יט צמאתי. מלשון צמאון: נאד. שם כלי מה:

כ ושאלך. מלשון שאלת מה: פה. בהאהל הזה:

כא יתד. מסמר: המקבת. שם כלי, ובו מכין ביתד לנעצו בארץ, וכן ( ישעיהו מד יב ) ובמקבות יצרהו: בלאט. בהסתר ובחשאי, כמו ( רות ג ז ) ותבא בלט ותגל מרגלתיו: ותתקע. נעצה ותחבה: ברקתו. הוא צד גובה הפנים, וכן ( שיר השירים ד ג ) כפלח הרמון רקתך, ונקראת כך למיעוט הבשר שבה, כמו ( בראשית מא יט ) ורקות בשר: ותצנח. ענין נעיצה, כמו ( לעיל א יד ) ותצנח מעל החמור: נרדם. מלשון תרדמה ושינה: ויעף. ענין יגיעה:

כד וקשה. רוצה לומר: וחזקה:

פרק-ה

א ותשר. מלשון שירה:

ב בפרע פרעות. ענין נקמה, כי ( במדבר לא ב ) נקום נקמת, תרגמו: אתפרע פרענותא:

ג האזינו. הטו אוזן לשמוע: רזנים. ענין שררה, כמו ( תהלים ב ב ) ורוזנים נוסדו יחד:

ד בצעדך. מלשון צעדה והלוך: נטפו. מלשון טפטוף ושפיכה: עבים. עננים:

ה נזלו. ענין נטיפה, כמו ( במדבר כד ז ) יזל מים מדליו:

ו חדלו. מנעו: ארחות. נתיבות, דרכים ושבילים: עקלקלות. מלשון מעוקל ומעוקם:

ז פרזון. ערי פרזי, מבלי חומה סביבה: שקמתי. מלשון קימה:

ח לחם. מלשון מלחמה: שערים. עיירות, כמו ( דברים יז ב ) באחד שעריך: מגן. תריס: ורמח. כידון:

ט לחוקקי. הגדולים יקראו 'מחוקקים', כי הם משימים החוקים:

י אתנות. נקבות החמורים: צחרות. לבנות, כמו ( יחזקאל כז יח ) וצמר צחר: מדין. מלשון דין: שיחו. דברו, כמו ( דברי הימים א טז ט ) שיחו בכל נפלאותיו:

יא מחצצים. מלשון חצים: משאבים. מלשון שאיבת המים: יתנו. ענין דבור, כי ( דברים ו ז ) ושננתם, תרגומו: ותתנון, וכן ( לקמן יא מ ) לתנות לבת יפתח: פרזונו. מלשון פרזון:

יב עורי. מלשון הערה והתעוררות: ושבה שביך. מלשון שביה:

יג ירד. ענין ממשלה, כמו ( תהלים קי ב ) רדה בקרב אויביך: שריד. שיור, כמו ( ירמיהו מב יז ) שריד ופליט: לאדירים. כמו באדירים, למ''ד במקום הבי''ת, וכן ( ויקרא כז ז ) ונפלו לפניכם לחרב, ומשפטו: בחרב:

יד מני. מן, והיו''ד נוספת: בעממיך. מלשון עם: משכים. כי הכתיבה עשויה במשיכת הקולמוס: שבט. רוצה לומר, קולמוס:

טו ושרי. כמו ושרים: ברגליו. עם הלוכו, ואחריו, וכן ( לעיל ד י ) ויעל ברגליו: בפלגות. ענין חלוקה, כמו ( בראשית י כה ) נפלגה הארץ: חקקי. מלשון חקיקה, וענינו מחשבה:

טז המשפתים. מלשון שפה, וענינו גבול, וכן תרגם יונתן: תחומין, וגם הוא תרגום של גבול, וכן ( שם מט יד ) רובץ בין המשפתים: שרקות. הונח על הקול הבאה בקבוץ השפתים, וכן ( ישעיהו ה כו ) ושרק לו, והוא שאול על כל המקנה:

יז יגור. יאסף, כמו ( תהלים נט ד ) יגורו עלי עזים: אניות. באניות: לחוף. לשפת הים, וכן ( דברים א ז ) בחוף הים:

יח חרף. ענין בזיון: מרומי. מלשון רם:

יט על מי. עד מי: בצע. ענין חמדה, כמו ( תהלים ל י ) מה בצע בדמי:

כ ממסלותם. מלשון מסילה ודרך:

כא גרפם. ענין כבוד וטאטא, כי ( שמות לח ג ): היעיס תרגם אונקלוס מגרופיתא, ורצונו לומר שטפם: תדרכי. מלשון דרך והלוך: נפשי. רוצה לומר, אני בעצמי: עוז. כמו בעוז:

כב הלמו. ענין הכאה, כמו ( ישעיהו מא ז ) הולם פעם: עקבי. הוא סוף הרגל: מדהרות. כן יקרא דריסת ודליגת הסוס בכח, כמו ( נחום ג ב ) וסוס דהר, וכפל המלה לרוב הדהרות: אביריו. מלשון אביר וחזק:

כג אורו. מלשון ארור: מרוז. שם מקום:

כה בספל. באגן, והוא כלי מה, וכן ( לקמן ו לח ) מלא הספל מים: אדירים. רוצה לומר, גדולים וחשובים: חמאה. הוא שומן החלב:

כו ליתד. הוא המסמר: להלמות. ענין הכאה: עמלים. מלשון עמל וטורח: מחקה ראשו. הסירה ראשו, בדברי רבותינו זכרונם לברכה ( בבא בתרא פט א ) אין מוחקין במקום שגודשין, ורצונו לומר אין מעבירין דבר מה על המדה, להסיר הגודש: ומחצה. ענין נגוע ומוכה, כמו ( תהלים קי ו ) מחץ ראש: וחלפה. ענין העברה מעבר לעבר, כמו ( ישעיהו ח ח ) וחלף ביהודה: רקתו. הוא המקום שאצל העין:

כז כרע. ענין נפילה על הברכים: שדוד. נבוז ונאבד, כמו ( שם לג א ) ואתה לא שדוד:

כח בעד. נגד: נשקפה. ענין הבטה, כמו ( בראשית כו ט ) וישקף אבימלך, ואמר בלשון גפעל, על כי גם היא נשקפה למביטים בה: ותיבב. ענין יללה, כי ( במדבר י ה ) תרועה, תרגם אונקלוס: יבבא, וקול התרועה היא כיללה: האשנב. הוא כעין חלון, כמו ( משלי ז ו ) בעד אשנבי נשקפתי: בשש. ענין איחור, כמו ( שמות לב א ) בשש משה: פעמי. צעדה והלוך, כמו ( תהלים פה יד ) וישם לדרך פעמיו:

כט תענינה. כל אחת תענה ותשיב:

ל רחם רחמתים. רוצה לומר, עלמה ושתי עלמות, וקראן בלשון בזיון, על שם הרחם: לראש גבר. רוצה לומר, לכל איש: רקמה. מעשה רוקם בהרבה מיני צבעים: רקמתים. רוצה לומר, בשתי פנים, בהעבר מזה ובהעבר מזה, וחלוקה זה מזה, והוא כמו מעשה חושב:

פרק-ו

ב ותעז. מלשון עז וחזק: המנהרות. ענין אורה, כמו ( איוב ג ד ) ואל תופע עליו נהרה, והם מערות ולהם חלון יבוא בו אור היום: המצדות. מבצרים חזקים, כמו ( שמואל ב ה ז ) מצדת ציון:

ג אם. כאשר:

ד יבול. צמח האדמה, כמו ( תהלים סז ז ) ארץ נתנה יבולה: מחיה. ענייני צורך חיי הבריות, כמו ( בראשית מה ה ) למחיה שלחני:

ה כדי ארבה. כארבה, ומלת 'די', יורה ענין רבוי:

ו וידל. מלשון דל ועני:

ז אדות. תרגומו: עסק:

ט לחציכם. ענין דחק:

יא האלה. שם אילן מה: חבט. ענין דישה במקל, כמו ( ישעיהו כח כז ) במטה יחבט קצח: בגת. מקום דריכת הענבים נקרא גת: להניס. להבריח:

יב החיל. ענין כח:

יג בי. ענין בקשה: מצאתנו. מצאה אותנו: נטשנו. עזבנו:

יד הלא. המלה ההיא יורה על הזרוז:

יח תמש. ענין הסרה, כמו ( יהושע א ח ) לא ימוש ספר התורה:

יט ואיפת. שם מדה בת שלש סאים: בסל. שס כלי מה: והמרק. הרוטב, כמו ( ישעיהו סה ד ) ומרק פגלים: בפדור. הוא דוד נחושת, כעין יורה, וכן ( במדבר יא ח ) ובשלו בפרור:

כ הלז. הזה, כמו ( בראשית כד סה ) מי האיש הלזה:

כא המשענת. המטה שנשען עליה: הצור. הסלע: ותאכל. ותשרף:

כב אהה. ענין צעקת יללה: על כן. על אשר:

כד עודנו. מלשון עוד:

כה פר השור. הכפל לתוספת ביאור, כמו ( ויקרא טז ג ) פר בן בקר:

כו המעוז. מלשון עת וחוזק, והוא הסלע: במערכה. מלשון עריכה וסדור:

כח נתץ. נשבר וכתות:
