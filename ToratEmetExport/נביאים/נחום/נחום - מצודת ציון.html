פרק-א

א האלקושי. שם עיר אלקוש:

ב קנוא. ענין כעס כמו הם קנאוני בלא אל ( דברים לב ): ונוטר. ענין שמירת איבה כמו לא תקום ולא תטור ( ויקרא יט ):

ג בסופה. כן יקרא רוח החזק: ובשערה. כמו בסערה בסמ''ך וגם הוא רוח חזק. אבק. עפר דק:

ד גוער. ענין צעקת נזיפה: החריב. ענין יובש כמו חרבו המים ( בראשית ח ): אמלל. ענין כריתה כמו כי אמלל אני ( תהלים ו ): לבנון. שם יער:

ה התמוגגו. מל' המגה והמסה: ותשא. ענין שרפה כמו שאו משאת ( ירמיה ו ):

ו נתכה. ענין התכה ושפיכה כמו ותתך חמתי ( שם מד ): והצורים. מל' צור וסלע: נתצו. מל' נתיצה ושבירה:

ז למעוז. ענין חוזק: חוסי. ענין בטחון:

ח ובשטף. ענין מריצת המים כמו כנחל שוטף ( שם מז ):

י סירים. מין קוצים כמו כקול הסירים ( קהלת ז ): סבוכים. ענין אחיזה וקליעה וכן סבכי היער ( ישעיה י ): וכסבאם סבואים. ענין שתיית משקים המשכרים כמו ונסבאה שכר ( שם כו ): אכלו. נאכלו ר''ל נשרפו: מלא. ענין השלמה ר''ל נשלם בשולו:

יא בליעל. בלי עול שמים ר''ל רשע:

יב שלמים. מל' שלום: וכן נגוזו. הוי''ו היא במקום אעפ''כ וכן הנמלים עם לא עז ויכינו בקיץ לחמם ( משלי ל ): נגוזו. מל' גזוזה וכריתה: וענתך. מל' עינוי וצער:

יג מטהו. ענינו בדי העול כמו מוסרות ומוטות ( ירמיה נז ): ומוסרותיך. הם קשורי רצועי העול: אנתק. ענין ההעתק ממקומו:

יד קלות. מל' קל:

פרק-ב

ב עלה. ענין סלוק כמו אל תעלני ( תהלים קב ): מפיץ. מל' הפצה ופזור: נצור. ענין שמירה כמו נוצר תאנה ( משלי כז ): מצורה. ערי מבצר כמו ויבן ערי מצורה ( ד''ה ב יד ): צפה. ענין הבטה כמו עמדי וצפי ( ירמיה מח ): אמץ. ענין חוזק:

ג שב. ענין השקט ומרגוע כמו בשובה ונחת ( ישעיה ל ): גאון. ענין רוממות וממשלה: בקקום בוקקים. ענין ריקות כמו גפן בוקק ( הושע י ): וזמוריהם. כן יקראו ענפי הגפן כמו ויכרתו משם זמורה ( במדבר יג ):

ד גבוריהו. גבוריו: מאדם. מל' אודם: מתלעים. ל' תולעת שני והיא צבע אדומה: פלדות. מל' לפיד אש בדרך ההפוך והוא העץ שהשלהבת נאחז בו: והברושים. ר''ל מקלות מאילני ברוש: הרעלו. ענינו סם הממית ודוגמתו יין תרעלה ( תהלים ס ) המטמטם את הלב מרוע תכונתו והוא מל' ערל בדרך ההפך שהוא ענין אוטם כמו ערלה אזנם ( ירמיה ו ):

ה יתהוללו. ענין שטות ושגעון כמו ויתהולל בידם ( ש''א כג ) כי מרוב המרוצה נראה כמשתגעים: הרכב. ר''ל אנשי הרכב: ישתקשקון. ענין השמעת קול כמו והשיקו היקבים ( יואל ב ): ירוצצו. מלשון ריצה ורדיפה:

ו אדיריו. גבוריו: והוכן. מלשון הכנה: הסוכך. מל' סכך ומכסה:

ז נמוג. מל' המגה והמסה ואמר בדרך שאלה:

ח והוצב. ר''ל דבר הנצב במקום אחד מבלי רגילות התנועה וההליכה: גלתה. מל' גלוי: הועלתה. מל' עליה: ואמהותיה. שפחותיה: מנהגות. מל' הנהגה והבאה: מתופפות. מל' תוף והוא שם כלי נגון:

ט כברכת. הוא מקום בנוי באבנים ובסיד ושם מתכנסים המים כמו בתעלת הברכה ( מלכים ב יח ): מימי. ר''ל מימים רבים: נסים. בורחים: מפנה. מל' הפנה וסבוב:

י בוזו. מל' בזה ושלל: קצה. סוף: לתכונה. מל' הכנה: כבד. מל' כבוד או הוא ענין טיאוט וגריפה כמו התכבד כילק ( לקמן ג ):

יא בוקה ומבוקה. ענין ריקות כמו גפן בוקק ( הושע י ) וכפל המלה לחוזק הענין: ומבולקה. ענין כריתה כמו ובולקה ועוה פניה ( ישעיה כד ): ופיק. ענין כשלון כמו ולא יפיק ( ירמיה י ): פארור. קדרה כמו ובשלו בפרור ( במדבר יא ):

יב מעון. מדור: לכפירים. לביא. שם משמות האריה: גור. כן יקרא האריה הקטן וכן גור אריה יהודה ( בראשית מ''ט ):

יג בדי. בהרבות כמו בדי שופר ( איוב ל''ט ): ומחנק. מלשון חנק: חוריו. מלשון חור ונקב היא המערה אשר יסתר שם האריה: ומעונותיו. מלשון מעון ומדור:

יד והבערתי. ענין שריפה: רכבה. קבוצת המרכבות: מלאככה. מל' מלאך ושליח:

פרק-ג

א הוי. כמו אוי: כחש. שקר: פרק. ענין השבר כמו ופרסיהן יפרק ( זכריה י''א ): ימיש. ענין הסרה כמו לא ימיש עמוד הענן ( שמות י''ג ):

ב שוט. שבט ושרביט כמו שוט לסוס ( משלי כ''ו ): אופן. גלגל המרכבה: דוהר. כן יקרא דליגת וקפיצת הסוס כמו מדהרות דהרות אביריו ( שופטים ה' ): מרקדה. מלשון רקוד וקפיצה:

ג פרש. כן יקרא הרגיל לרכוב על הסוס: ולהב. כן יקרא ברזל החרב כי כשהיא מלוטשת מברקת כלהב אש וכן ולהב חניתו ( ש''א י''ז ): וברק. כן יקרא ברול החנית כי הוא מבריק כברק כאשר ילטשו וכן ברק חרבי ( דברים ל''ב ): וכובד. ענין רבוי כמו בעם כבד ( במדבר כ' ): פגר. כן יקרא גוף ההרוג: לגויה. לגוף כמו גוית שאול ( ש''א ל''א ):

ה וגליתי. מלשון גלוי: שוליך. הוא תחתית הבגד כמו על שולי המעיל ( שמות כ''ה ): מערך. גלויך וכן עריה בושת ( מיכה א' ): קלונך. מלשון קלון ובזיון:

ו שקוצים. מלשון שקץ ומאוס: ונבלתיך. ענין כיעור ומגונה כמו אגלה את נבלותה ( הושע ב' ): כראי. ענין טנוף ולכלוך וכן מוראה ונגאלה ( צפניה ג' ):

ז ידוד. מלשון נדידה: שדדה. ענין עושק: ינוד. מלשון נדידה והנעה: מאין. מאיזה מקום:

ח התיטבי. מלשון טוב: מנוא. נוא היא אלכסנדריא כן ת''י: אמון. כמו המון והוא ענין עם רב וכן ואת יתר האמון ( ירמיה כ''ב ): ביאורים. כן יקראו החפירות המושכות מים: חיל. הוא החומה הנמוכה שלפני הגבוה וכן ויאבל חיל וחומה ( איכה ב' ):

ט עצמה. ענין חוזק: פוט ולובים. שמות אומות:

י עולליה. בנים הקטנים: ירטשו. ענין בקיעה כמו וקשתות נערים תרטשנה ( ישעיה י''ג ): ידו. ענין השלכה כמו ואל עמי ידו גורל ( יואל ד ): רתקו. ענין שלשלאות כמו עשה הרתוק ( יחזקאל ז ) ור''ל קשרו בשלשלאות: בזקים. גם הוא ענין שלשלאות כמו לאסור מלכיהם בזיקים ( תהלים קמט ) ועם כי אמר רתקו אמר גם בזקים לתוספת ביאור וכן מסעף פארה ( ישעיה י ):

יא נעלמה. מל' העלם: מעוז. ענין חוזק:

יב עם בכורים. כמו ובכורים וכן אדמוני עם יפה עינים ( ש''א טו ) ור''ל ויפה עינים: בכורים. ענין גמר הבשול כמו בכורה אותה נפשי ( מיכה ז ): ינועו. מל' הנעה והנדה:

יג בריחיך. כעין מטה לנעול השערים ע''י:

יד מצור. כן יקרא העיר בעת שהגייסות צרים עליה: שאבי. כן יקרא לקיחת המים כמו ותשאב לכל גמליו ( בראשית כד ): ורמסי. מל' רמיסה ודריכה: בחומר. הוא הטיט וכפל הדבר במ''ש: החזיקי. ענין אחיזה: מלבן. מל' לבינה העומדים לבנין החומה:

טו כילק. מין ארבה: התכבד. ענין טיאוט וגריפה לנקות ובדרו''ל מכבדים את הבית ( ברכות נה ) וכן כבוד מכל כלי חמדה ( לעיל ב ):

טז הרבית. מל' רבוי: רוכליך. סוחריך וכן בעיר רוכלים שמו ( יחזקאל יז ): פשט. ענין פרישה: ויעוף. מל' עפיפה ופריחה:

יז מנזריך. מל' נזר ועטרה כמו את הנזר ( מ''ב יא ): וטפסריך. ענין שררה כמו פקדו עליה טפסר ( ירמיה נא ): כגוב גובי. הוא מין ארבה וכן יוצר גובי ( עמוס ז ) ור''ל מין הגדול שבארבה ודוגמתו בעדי עדיים ( יחזקאל טז ) ור''ל בהטוב והיפה שבעדים: בגדרות. מל' גדר וכותל: קורה. מל' קור וצנה: נודד. מל' נדידה: אים. איה הם:

יח נמו. מל' תנומה ושינה: רועיך. מנהיגיך: אדיריך. גבוריך: נפושו. הוא מל' נופש ומרגוע כמו שבת וינפש ( שמות לא ):

יט כהה. ענין עכירות וחושך כמו ככה הנגע ( ויקרא יג ): נחלה. מל' חולי: תקעו כף. ענין ההכות כף אל כף לשמחה וכן כל העמים תקעו כף ( תהלים מז ):
